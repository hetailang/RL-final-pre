# RL Final Presentation

This folder contains the Slidev deck for the final course-project presentation.
It is separated from `midterm_report/` so the midterm and final materials can
evolve independently.

## Files

```text
slides.md           # Final presentation source
slides-export.pdf   # Exported PDF, generated by npm run export
package.json        # Slidev scripts and dependencies
package-lock.json   # Locked npm dependency versions
```

## Preview

```bash
cd final_report
npm install
npm run dev
```

Slidev will print a local URL, usually `http://localhost:3030`.

## Export

```bash
cd final_report
npm run export
```

The exported PDF is written to `slides-export.pdf`.
